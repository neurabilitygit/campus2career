# Parent Briefs and Scenario Chat

## Parent brief generator
Inputs:
- scoring output
- accomplishments
- target goal
- upcoming deadlines
- parent-visible insights

Output:
- direct but measured monthly brief for a parent-first buyer

## Scenario chat
Inputs:
- scoring output
- target role family
- sector cluster
- communication style
- scenario question
- visible insights

Output:
- blunt strategic guidance for the student
- ends with concrete next actions
