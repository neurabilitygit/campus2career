# Source Notes

O*NET Web Services provides API access to the entire O*NET Database and supports occupation search and detailed occupational data retrieval.

The BLS Public Data API supports JSON access to published historical time series data. Version 2.0 supports up to 50 series per request and up to 20 years of data, with registration for higher-use features.

Implementation note:
The starter repo includes scaffolding, not a finished production ETL. Exact endpoints and BLS series IDs should be locked in during buildout.
